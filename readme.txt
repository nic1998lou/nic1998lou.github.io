HTML5 Stereo Viewer Ver1.7 (2022/02/26)
Fixed an issue that prevented the display of 3DLCDs with a width of 2048 pixels or more.

Ver1.6 (2021/11/21)
Add recording function.
Press "Rec" button at the top of the screen to start recording, "Stop" button to stop recording, and "DL" to download the recording file.
The recording file is in webm format, so if you want to convert it to mp4, download ffmpeg.exe and store it in this folder, then drop the webm video file into "webmtomp4.bat" and it will be converted to mp4 file.

ffmpeg.exe
https://ffmpeg.org/

録画機能を追加
画面上部の「Rec」ボタンを押すと録画スタートし、「Stop」ボタンを押すと録画を終了、「DL」で録画ファイルをダウンロードします。

録画ファイルは、webm形式なので、mp4に変換したい場合は、ffmpeg.exeをダウンロードし、このフォルダに格納し、「webmtomp4.bat」にwebm動画ファイルをドロップすると、mp4ファイルに変換されます。
